## 🌟 Overview

**Aone Digital India** is a premium, production-ready showcase website for a Mobile Phones & Home Appliances retail brand. The website is designed to:

- Showcase top mobile and appliance brands
- Generate leads through contact forms and WhatsApp integrations
- Deliver a world-class user experience with smooth animations
- Be fully SEO-optimized, accessible, and performance-first
- Support dark/light themes with a glassmorphism design system

## 🚀 Getting Started

### Prerequisites

- Node.js >= 20.x
- npm >= 10.x or pnpm >= 9.x
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/your-org/aone-digital-india.git

# Navigate to project directory
cd aone-digital-india

# Install dependencies
npm install
# or
pnpm install

# Copy environment variables
cp .env.example .env.local

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📜 Available Scripts

```bash
npm run dev          # Start development server
npm run build        # Build for production
npm run start        # Start production server
npm run lint         # Run ESLint
npm run lint:fix     # Fix ESLint errors automatically
npm run format       # Format code with Prettier
npm run type-check   # TypeScript type checking
npm run test         # Run unit tests
npm run test:e2e     # Run end-to-end tests
npm run analyze      # Analyze bundle size
```
